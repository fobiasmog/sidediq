# Sidediq
ZeroMQ Rails Active Job backend

Why? Just for fun


## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem "sidediq"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install sidediq
```
